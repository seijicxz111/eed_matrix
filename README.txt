Open through XAMPP:
1. Copy the EED_MATRIX folder to C:\xampp\htdocs\
2. Start Apache in XAMPP.
3. Visit: http://localhost/EED_MATRIX/

Do not open index.html directly with file://. Browser security blocks the certificate PDF from loading under file://.

Files:
- index.html = HTML structure
- css/styles.css = all styling
- js/app.js = all JavaScript behavior
- assets/osas_logo.png = OSAS logo
